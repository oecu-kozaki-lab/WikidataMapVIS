# WikidataMapVIS
Wikidataのクエリサービス（SPARQLエンドポイント）を使った，経路検索付きのMAPを簡単に生成するサービスです．  
MAPに表示したいデータの **「具体例」** を検索し，そのデータのWikidata上での **「分類」** を選択することで， **「いろいろな分類のMAP」** が生成できます．

サービスの実行は **[https://wd-map.hozo.jp/](https://wd-map.hozo.jp/)** から
